Form有关问题

发表于 2018-03-24

form表单有什么作用？有哪些常用的input 标签，分别有什么作用？

  1. form元素是表单元素的前端处理器，负责数据的收集、打包和发送。
  2. 常用的input标签有：
    • text 定义单行文本框
    • password 定义密码域
    • checkbox 定义复选框
    • radio 定义单选按钮
    • file 定义文件域
    • submit 定义提交按钮
    • reset 定义重置按钮

post和get 式的区别？

  • get方法是从服务器获取数据，而post是向服务器上传数据
  • get方法是将表单中的数据以“名/值”对的形式添加到action所设置的URL后面，并使用“？”符号连接URL，各变量之间使用“&”连接

在input里，name有什么作用？

定义表单元素的名称后台服务器能够利用该属性来读取其中的数据

radio如何分组?

设置name属性，设置为相同的一组

placeholder属性有什么作用?

加入提示性的语句

type=hidden隐藏域有什么作用?

如果有数据需要作为参数传递给服务器，使用隐藏域避免数据对表单交互行为的干扰
