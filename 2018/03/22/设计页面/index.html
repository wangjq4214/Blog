元素分类

结构语义

  • div： 定义一块区域
  • span： 在文本中定义区域
  • ol： （ordered）无序列表
  • ul： （unorder）有序列表
  • li： 每条列表项
  • dl： 定义列表
  • dt： 定义词条
  • dd： 定义解释
  • p: 段落

内容语义

  • a： 超链接
  • abbr： 定义缩写
  • address： 定义地址
  • var： 定义变量
  • pre： 预定义格式文本
  • blockquote： 区块引用
  • cite： 定义文本
  • q： 引用短语
  • strong： 定义重要文本
  • em： 定义重要文本

修饰语义

  • b： 粗体文本
  • i： 斜体文本
  • big： 大文本
  • small： 小文本
  • sup： 上标
  • sub： 下标
  • bdo： 定义显示方向
  • br： 换行
