Form有关问题

发表于 2018-03-24

form表单有什么作用？有哪些常用的input 标签，分别有什么作用？

  1. form元素是表单元素的前端处理器，负责数据的收集、打包和发送。
  2. 常用的input标签有：
    • text 定义单行文本框
    • password 定义密码域
    • checkbox 定义复选框
    • radio 定义单选按钮
    • file 定义文件域
    • submit 定义提交按钮
    • reset 定义重置按钮

post和get 式的区别？

  • get方法是从服务器获取数据，而post是向服务器上传数据
  • get方法是将表单中的数据以“名/值”对的形式添加到action所设置的URL后面，并使用“？”符号连接URL，各变量之间使用“&”连接

在input里，name有什么作用？

定义表单元素的名称后台服务器能够利用该属性来读取其中的数据

radio如何分组?

设置name属性，设置为相同的一组

placeholder属性有什么作用?

加入提示性的语句

type=hidden隐藏域有什么作用?

如果有数据需要作为参数传递给服务器，使用隐藏域避免数据对表单交互行为的干扰

元素分类

结构语义

  • div： 定义一块区域
  • span： 在文本中定义区域
  • ol： （ordered）无序列表
  • ul： （unorder）有序列表
  • li： 每条列表项
  • dl： 定义列表
  • dt： 定义词条
  • dd： 定义解释
  • p: 段落

内容语义

  • a： 超链接
  • abbr： 定义缩写
  • address： 定义地址
  • var： 定义变量
  • pre： 预定义格式文本
  • blockquote： 区块引用
  • cite： 定义文本
  • q： 引用短语
  • strong： 定义重要文本
  • em： 定义重要文本

修饰语义

  • b： 粗体文本
  • i： 斜体文本
  • big： 大文本
  • small： 小文本
  • sup： 上标
  • sub： 下标
  • bdo： 定义显示方向
  • br： 换行

网页设计基础

发表于 2018-03-22

开始系统的学习HTML5的第四天，今天创建第一篇markdown。

网页类型

根据是否使用了服务器技术，人们把网页分为静态网页和动态网页。

  1. 静态网页
    • 是实实在在保存在服务器上的文件，每个网页都是独立的文件
    • 网页内容相对稳定
    • 没有数据库的支持
    • 交互性差
  2. 动态网页
    • 以数据库为基础
    • 可实现更多的网页功能
    • 只有请求网页时，才会返回完整的网页

网页构成

  • 文本
  • 图像
  • 动画
  • 连接
  • 导航条
  • 表格
  • 表单
  • 其他
